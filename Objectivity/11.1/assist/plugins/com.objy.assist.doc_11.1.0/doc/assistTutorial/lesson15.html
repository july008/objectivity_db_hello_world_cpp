Lesson 15: What’s Next?
Congratulations! You finished the basic Objectivity/DB tutorial. In this tutorial, you learned how to use Objectivity/Assist to browse an Objectivity/DB federated database. That is, you learned:
■
How to create an Assist project for an existing federated database.
■
How to browse the federated database’s catalog to see the various file and process resources that comprise the federation.
■
How to browse the federated database’s data to see the persistent objects within the storage hierarchy of containers and databases.
■
How to find persistent objects using Objectivity/DB’s predicate query language.
■
How to browse the federated database’s schema to see the types of data that can be stored in the federation.

Read the documentation
You can also explore the documentation set in your Objectivity/DB installation directory.
Most books are delivered in PDF, which you can access by opening:
installDir/doc/ObjyBooks.pdf
Some books are also delivered in HTML, which you can access by opening:
installDir/doc/java/index.html
installDir/doc/objyHelp/index.html
On Windows, you can click the following, and select from the menu:
Start > Programs (or All Programs) > Objectivity x.x > Documentation
