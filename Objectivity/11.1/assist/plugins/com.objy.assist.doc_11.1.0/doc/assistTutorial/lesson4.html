Lesson 4: What is Objectivity/Assist?
Objectivity/Assist (also called Assist) is a tool for viewing and managing the data, files, and processes of an Objectivity/DB federated database. For example, a developer might use Assist to see whether an application populated a federated database correctly. Or, a database administrator could use Assist to determine where a federated database’s files and processes are located.
This tutorial uses Assist to give you a guided tour of basic Objectivity/DB components.
Note: Assist is implemented as a rich client application on the Eclipse Rich Client Platform (RCP).
Assist at Startup
The following shows how Assist appears when started for the first time.
Objectivity/Assist displays the FD Project Explorer, an editor area, an Outline area, and the welcome screen.
Note: If you are running Assist on a Macintosh, you will see some minor differences in the display.
During this tutorial, you will use the FD Project Explorer to open various Objectivity/DB components in an editor.
The Outline area will not be used in this tutorial. To close it, click on the X in the top right of this tab. Later, you can restore the outline tab by clicking Window > Show Outline View.
