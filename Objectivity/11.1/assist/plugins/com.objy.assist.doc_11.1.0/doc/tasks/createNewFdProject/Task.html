Creating an FD Project using a new FD


Make sure you are in the FD Project Explorer Perspective. If not select the Window pull-down menu, select Open Perspective, select Objectivity/Assist. This will open the FD Project Explorer Perspective. Create a New FD Project by selecting File/New/Project from the File pull-down menu. Select Objectivity/Assist and then FD Project and hit the Next button.

 Follow the instructions. When you get to the FD Project Specification window select New FD, and enter Example4.1 in the System Name field, and browse for the System DB Dir in installdir\Examples\Example4.1\Data, set the FD Number field to 141, and hit Finish.
