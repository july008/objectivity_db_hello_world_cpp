Objectivity/C++ Tutorial
Introduction
This tutorial introduces Objectivity/DB and explains how to get started building C++ programs that use Objectivity/DB. You will learn how to build simple programs that use Objectivity/DB and will then explore more advanced features. This tutorial is intended for use by developers with little or no database experience but who are familiar with C++ programming concepts.
This tutorial focuses on the development of Objectivity/C++ applications and the use of the visualization tool Objectivity/Assist. While the tutorial was created to be mostly platform agnostic, there are Visual Studio project files and makefiles available in the example directory to compile and run the example.
It is assumed you have completed the Objectivity/DB Basics Tutorial.
Setting Up
You will learn how to set up the Visual C++ IDE and Objectivity/Assist for development of Objectivity/C++ applications.
Lesson 1: Getting Started
You will compare a native C++ program with a similar program written for Objectivity/DB, and you will use Objectivity/Assist to inspect the federated database.
Lesson 2: Connections and Sessions
You will start to develop your Objectivity/C++ application. In this lesson you will become familiar with Objectivity/C++ connections and sessions, which set up and control access to an Objectivity/DB federated database and its contents.
Lesson 3: Creating Persistent Objects
You will learn how to create Objectivity/DB objects using managed placement. In addition, you will learn one way of discovering these objects.
Lesson 4: User-Defined Classes
You will learn how to define your own classes in the Objectivity/DB schema.
Lesson 5: Working with User-Defined Classes
You will learn how to create instances of user-defined classes, and how to subsequently find these instances and delete them.
Lesson 6: Defining Associations
You will learn how to define Objectivity/C++ associations between user-defined classes and create associations between the object instances.
Lesson 7: Managed Placement
This brief introduction to managed placement will show you how to customize your default placement model.
Lesson 8: Searching and Indexing
You will create a predicate query and learn how to create an index to improve query performance.
