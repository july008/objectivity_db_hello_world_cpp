Configuring Your Installation

  1. Set up your license file.

    • If necessary, manually install the license file and upgrade any existing federated databases.

  2. Set environment variables for developers.

    • Refer to installDir/setup script, then modify the appropriate environment file for your platform.

  3. Set up servers.

    • If not set by the installer, create a log directory for the lockserver.
    • Verify that the lock server is running using oocheckls from a terminal, starting it if necessary using oolockserver.
    • Optionally start the advanced multithreaded server (AMS) or the query server.
    • Optionally configure servers to start automatically at boot time.
    • For security purposes, consider creating special-purpose group accounts (with minimal required permissions) under which to run servers.

  4. Validate your install.

    • Run the Java or C++ helloWorld sample in your installDir/samples directory.

Note: For additional configuration information, review the appropriate configuration documentation for your product selection.
